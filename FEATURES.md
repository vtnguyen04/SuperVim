# 🎨 SuperVim Enhanced Interface Features

## ✨ What's New in SuperVim 2.1

SuperVim now features a **stunning, Hyprland-style interface** with advanced tab management, beautiful animations, and professional layouts that rival modern IDEs.

## 🪟 **Advanced Tab Management**

### Beautiful Bufferline
- **Slanted tabs** with Tokyo Night styling
- **Live diagnostics** displayed in tabs
- **Smart grouping** and tab organization
- **Pin/unpin buffers** for persistent access

### New Keybindings:
- `<leader>bp` - Pin current buffer
- `<leader>bo` - Close all other buffers
- `<leader>br` - Close buffers to the right
- `<leader>bl` - Close buffers to the left
- `<leader>bg` - Close current buffer group
- `Shift + H/L` - Navigate between tabs

## ⌨️ **New Keybindings**

### UI Operations (`<leader>u`)
| Key | Action | Description |
|-----|--------|-------------|
| `<leader>uz` | Zen Mode | Distraction-free coding |
| `<leader>ut` | Twilight | Focus mode with dimming |
| `<leader>uw` | Pick Window | Visual window selection |
| `<leader>um` | Minimap | Toggle code minimap |
| `<leader>uf` | Fold Preview | Preview collapsed code |
| `<leader>un` | Clear Notifications | Dismiss all notifications |

### Terminal Operations (`<leader>t`)
| Key | Action | Description |
|-----|--------|-------------|
| `<leader>tf` | Float Terminal | Beautiful floating terminal |
| `<leader>th` | Horizontal Terminal | Split terminal below |
| `<leader>tv` | Vertical Terminal | Split terminal right |
| `<leader>tn` | Node REPL | JavaScript/Node.js console |
| `<leader>tp` | Python REPL | Python interactive shell |
| `<C-\>` | Quick Terminal | Toggle terminal overlay |

### Buffer Management (`<leader>b`)
| Key | Action | Description |
|-----|--------|-------------|
| `<leader>bp` | Pin Buffer | Keep buffer always visible |
| `<leader>bo` | Close Others | Close all other buffers |
| `<leader>br` | Close Right | Close buffers to the right |
| `<leader>bl` | Close Left | Close buffers to the left |
| `<leader>bg` | Close Group | Close current buffer group |
